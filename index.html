<button type="button" class="btn btn-default" data-toggle="push" data-target="#sidebar">

	</span>

Usage

Via data attributes

Activate any offcanvas element without writing JavaScript. Set data-toggle="push" on a controller element. Such as a button or a link, along with the data-target="#sidebar" attribute. 

<button type="button" class="btn btn-default" data-toggle="push" data-target="#sidebar">
	<span class="glyphicon glyphicon-hamburger-menu"></span>
	<span class="sr-only">
		Toggle navigation
	</span>
</button>

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-example="x". Here is a practical demonstration, on how to parse custom options. To your specific element: data-duration="500"

Name type default description
easing string cubic-bezier(.2,.7,.5,1) Easing method, used for the element when it (opens / closes).
duration integer 300 Duration for an element to (open / close)
delay integer 0 Delay before the element (opens / closes)
distance integer 250 Distance an element (opens / closes)
antiScroll boolean true Enable or disable Anti Scrolling (on the canvas element)
keyboard boolean true Enable or disable keyboard closing (escape key to close the element)
overlay boolean true Enable or disable a modal like overlay (outside the element).
canvas string #canvas The canvas element covers, the pages content (except for the sidebar elements).

Events

Bootstrap's push class exposes a few events for hooking into offcanvas functionality.

Event type description
open.bs.push Trigger the open sequence.
opened.bs.push Indicate that the opening sequence is complete.
close.bs.push Trigger the close sequence.
closed.bs.push Indicate that the closing sequence is complete.
click.bs.push If the user clicks on the canvas element, call the close functionality.
keydown.bs.push Add or remove keyboard functionality to close a sidebar, with a specified keypress.
touchmove.bs.push Turn on or off the anti touch functionality for touch devices.
